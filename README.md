Reflection:

I started this project with the goal of making a teleport menu that allowed somebody to teleport to differnt places on the map. Overall, the process wasn't too bad. Because C sharp, the language I wrote the mod in, and Java are overall similar in concept
I was able to pretty much think about it like java and just google and use websites to figure out how to write the equivalent in C sharp. I used programiz: https://www.programiz.com/csharp-programming/arraylist, w3schools https://www.w3schools.com/CS/index.php along with just general googling
There are a lot of things I inherited from Stardew valley, ex I inherited IClickableMenu from StardewValley.Menus (which is the default base class menu for the teleport menu, as well as Mod from StardewModdingAPI (ofc since its a stardew mod)
I did fail quite a couple of times. When I was making the original teleport menu, I was a bit slow and dind't realize that I had to click build in rider for it to actually work on SDV, so that took a while to learn. 

Sources for locations of teleports: https://stardewvalleywiki.com/Pelican_Town as well as some from the CJB cheats menu https://www.nexusmods.com/stardewvalley/mods/4 for teleport locations
Waypoint logic utilizes pulling coordinates from in-game, something I learned how to do from the stardew modding wiki.
